
# UnicodeSparklines

Generate [sparkline](https://en.wikipedia.org/wiki/Sparkline) plots in R
with [Unicode character symbols](https://en.wikipedia.org/wiki/Unicode).

``` r
remotes::install_github("log-linear/UnicodeSparklines")
```
